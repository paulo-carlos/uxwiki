CCFK UI Component Library

Client Framework (CCFK) is a library of UI components for developing modern web-based applications that are styled to meet Nokia branding standards. Built using ReactJS, the CCFK library provides a wide range of functionality for building enterprise-level web applications.

CCHT Charts

CHT is a react-based UI library for creating visualizations for web-based applications. CCHT is based on a commercial license for HighCharts.

CMAP Client Framework

CMAP Client provides a UI library of FreeForm UX styled openlayers components can be leveraged in constructing larger web-based applications. Also see CMAP Server.

CRPT Reporting

CRPT provides deployment options for Jaspersoft reporting tools.
